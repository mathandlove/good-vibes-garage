## Project Architecture
React + Vite + TypeScript frontend. CSS Modules (not Tailwind).
Pages: `src/pages/`. Components: `src/components/`.
Images: `src/assets/Images/[slug]/` as WebP. OG images: `public/` as JPG.
Routes defined in `src/main.tsx`.
Production domain: `https://goodvibesgarage.ai`.

## General Rules
- Before editing any file, state which exact file you're targeting.
- If multiple versions of a file might exist, confirm before editing — don't guess.
- When I report a problem, fix it. Don't report findings back as a status update.
- Don't ask for specs or confirmation when the task is clear from context.
- Be strict in audits — flag every violation, prefer false positives over misses.
- Do not hallucinate external data, backlinks, or search results.
