# Empower Play — LEGO Book Prototypes

**Source URL:** https://www.buildempathy.com/empowerplay

## Overview

This page describes mPath's work developing children's books designed to encourage play. The organization prototyped and tested "30 different books, organized chronologically with some tested simultaneously" in an effort to encourage play.

---

## Content Structure

The page contains **30 linked resources**, primarily Google Drive file links to book prototypes, with one external link to wonder.io. These books represent the full iteration sequence tested throughout mPath's LEGO development process.
